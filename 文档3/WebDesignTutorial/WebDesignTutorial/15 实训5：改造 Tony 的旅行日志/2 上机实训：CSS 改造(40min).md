## 2.7 设置文本装饰线

我们先完成：

>[!info] 任务 11：
>  为 h1 h2 元素增加下划线。

然后删除该样式后再尝试下面这段代码：

```css
	h1,h2 {
		border-bottom: thin dotted #888888;
	}
```

你是否能发现哪种设置文本装饰线的方法更灵活，能否得出我们应该采用哪种形式为文本设置装饰线的结论呢。
